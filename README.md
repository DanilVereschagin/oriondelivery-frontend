### Установка

`npm i`

### Перед установкой:
- Заполните файл .env по подобию .env.sample
- заполните папку assets картинками блюд и ингредиентов

### Запуск в режиме разработки

`npm run dev`

### Запуск сборки приложения

`npm run build`

### Запуск собранной версии

`npm start`
